Dependencies:
python3
django
south
django-registration
django-bootstrap3
django-cron 

Create a virtualenviroment for a python 3 version:
$ virtualenv -p /usr/bin/python3.4 env

Activate the enviroment (NOT as root):
$ source ./env/bin/activate

Install dependecies:
$ pip install -r requirements.txt

Create initial database migrations for the following apps:
userprofile
groupaccount
transaction
transactionreal
groupaccountinvite

$ python3.3 manage.py schemamigration <appname>

then create the database, and create a root user (you have to create a userprofile for this first user later via the admin)
$ python3.3 manage.py syncdb

migrate all apps
$ python3.3 manage.py migrate <appname>


Run test server by running,
$ python manage.py runserver <ip>:<port>

Create a userprofile for the first root user,
- go to admin /admin/
- create new user profile and link it to the root user
